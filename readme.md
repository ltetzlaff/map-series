# mapseries

Handle an array of promises sequentially like Bluebird.mapSeries

## Usage

### NodeJS

```js
const { mapSeries } = require("mapseries")
```

### Typescript

```js
import { mapSeries } from "mapseries"
```

with `tsconfig.json`:

```json
...
"compilerOptions": {
  "moduleResolution": "node"
  ...
```
